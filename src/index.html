SmartCalc_v2.0

Этот калькулятор позволяет выполнять четыре основные математические операции и десять математических функций. Для использования калькулятора необходимо ввести два числа и выбрать оператор, затем нажать кнопку "=". Для построение графиков необходимо задать область определения и область значения функций ограничиваются по крайней мере числами от -1000000 до 1000000 и задать график функциии , заданной с помощью выражения в инфиксной нотации с переменной x. После этого нажать на "graf".

Особенности калькулятора:

  • кнопка "sin" вычисляет синус
  • кнопка "cos" вычисляет косинус
  • кнопка "tan" вычисляет тангенс
  • кнопка "asin" вычисляет арксинус
  • кнопка "acos" вычисляет арккосинус
  • кнопка "atan" вычисляет акртангенс
  • кнопка "sqrt" вычисляет корень квадрата
  • кнопка "ln" вычисляет натуральный логарифм
  • кнопка "log" вычисляет десятичный логарифм
  • кнопка "mod" вычисляет деление с остатком
  • Выполнение четырех основных математических операций
  • Возможность работать с десятичными числами

Данный калькулятор создан с использованием языка С/С++.
